# **Predictive Analysis Using Machine Learning**



## Overview

This project implements a predictive analysis model using Logistic Regression
to predict passenger survival based on the Titanic dataset.



#### Dataset

Titanic dataset (loaded using seaborn)



#### Model Used

Logistic Regression (Classification)



#### Steps Covered

Data loading and cleaning

Feature selection

Train-test split

Model training

Model evaluation



#### Results

The model achieved good accuracy and demonstrated effective predictive capability.
